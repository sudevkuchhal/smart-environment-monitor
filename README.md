# 🌍 Smart Environment Monitoring System

An IoT based Smart Monitoring System built using ESP32, DHT22 Sensor, OLED Display and Blynk IoT Platform.

## 🚀 Features

- 🌡 Real-Time Temperature Monitoring
- 💧 Humidity Detection
- 📏 Distance Measurement using Ultrasonic Sensor
- 🚨 Smoke Detection & Alert System
- 📱 IoT Dashboard using Blynk
- 🖥 OLED Live Data Display
- 🔔 Buzzer & LED Alert System

---

## 🛠 Technologies Used

- ESP32
- Arduino IDE
- Embedded C++
- Blynk IoT
- DHT22 Sensor
- Ultrasonic Sensor
- OLED Display

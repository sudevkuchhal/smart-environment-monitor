# 🚀 SMART MONITOR X — ESP32 IoT Monitoring & Alert System

A feature-rich IoT monitoring platform built using ESP32 that combines real-time environmental sensing, intelligent alert mechanisms, cloud connectivity, and a futuristic OLED user interface.

Designed to showcase Embedded Systems, IoT, Automation, and Real-Time Data Visualization in a single integrated solution.

---

## ✨ Key Features

### 📊 Real-Time Monitoring

* 🌡 Temperature Monitoring
* 💧 Humidity Monitoring
* 📏 Ultrasonic Distance Measurement
* 🚨 Gas & Smoke Detection

### 🖥 Interactive User Interface

* SH1106 OLED Display
* JARVIS-Style Boot Animation
* Radar Scanner Animation
* Heartbeat Status Indicator
* Live Sensor Visualization

### 🌐 IoT Connectivity

* Blynk Cloud Integration
* Real-Time Remote Monitoring
* Web Dashboard Support
* Smart Wi-Fi Connection Management

### 🔔 Intelligent Alerts

* Email Notification System
* Buzzer Alert Mechanism
* Visual Status Indicators
* Emergency Detection Response

### ⚡ Advanced Features

* OTA (Over-The-Air) Updates
* Multi-Sensor Integration
* Real-Time Analytics
* Optimized ESP32 Performance

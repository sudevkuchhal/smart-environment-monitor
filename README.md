# Smart Environment Monitoring System 🌍

An IoT based project using ESP32, DHT22 sensor, OLED display and Blynk IoT platform.

## ✨ Features
- Temperature Monitoring
- Humidity Monitoring
- Distance Detection
- Real-Time IoT Dashboard
- OLED Display Output
- Smart Automation

## 🛠 Technologies Used
- ESP32
- Arduino IDE
- Blynk IoT
- C++
- Embedded Systems
